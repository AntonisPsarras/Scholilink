# Contributing to ScholiLink

> **This project is not accepting contributions.**

ScholiLink is an **archived portfolio piece**, published to showcase architecture and code quality. It is provided as-is, with no active maintenance, roadmap, or community support.

## Pull Requests

Pull requests will **not** be reviewed or merged. Please do not open them.

## Issues

The issue tracker is disabled (or ignored). Bug reports and feature requests will not be acted upon.

## Forks

You are warmly encouraged to **fork this repository** and do whatever you like with the code under the terms of the [MIT License](LICENSE). The codebase is intentionally well-documented and structured to make it easy to learn from, adapt, and extend for your own projects.

No support will be provided for forked projects.

## Why?

This repository exists strictly as a static, read-only portfolio artifact. The author has no intention of maintaining an open-source community, responding to tickets, or accepting outside contributions. Thank you for understanding.
